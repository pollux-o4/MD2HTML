# ADR-0040: E2E Playwright

## Status

Accepted (2025-03-13)

## Context

이 ADR 은 "E2E Playwright" 와 관련된 결정을 기록한다.
구체적인 배경은 사내 회의록 meeting-0040 를 참조.
스코프는 운영/도구/프로세스 영역이며, ADR-0001~0005 의 핵심 코드 결정과는
직접적인 코드 변경을 동반하지 않는다.

## Decision

E2E Playwright 를 표준으로 채택한다. 세부 운영 규칙은 별도 문서.

## Consequences

운영 일관성 확보. 신규 인력 온보딩 비용 감소.
구체적인 src/ 하위 파일 변경은 거의 없음.
