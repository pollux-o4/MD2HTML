# ADR-0039: Storybook 도입

## Status

Accepted (2025-10-10)

## Context

이 ADR 은 "Storybook 도입" 와 관련된 결정을 기록한다.
구체적인 배경은 사내 회의록 meeting-0039 를 참조.
스코프는 운영/도구/프로세스 영역이며, ADR-0001~0005 의 핵심 코드 결정과는
직접적인 코드 변경을 동반하지 않는다.

## Decision

Storybook 도입 를 표준으로 채택한다. 세부 운영 규칙은 별도 문서.

## Consequences

운영 일관성 확보. 신규 인력 온보딩 비용 감소.
구체적인 src/ 하위 파일 변경은 거의 없음.
